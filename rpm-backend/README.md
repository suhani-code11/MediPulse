\# MediPulse



MediPulse is a Flask-based Remote Patient Monitoring System that analyzes patient vital signs and provides health risk assessment.



\## Features



\- Heart Rate Analysis

\- SpO₂ Analysis

\- Temperature Analysis

\- Risk Prediction (Low, Medium, High)

\- Health Recommendations

\- PDF Report Generation

\- User-Friendly Web Interface



\## Technologies Used



\- Python

\- Flask

\- HTML

\- CSS

\- Jinja2 Templates



\## Project Structure

rpm-backend/

\## Installation



```bash

pip install -r requirements.txt

python app.py

```
